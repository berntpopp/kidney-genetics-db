# Administrator Guides

**Guides for system administrators and data curators**

## Documentation in This Section

- **[Admin Panel](admin-panel.md)** - Admin interface usage
- **[User Management](user-management.md)** - Managing users and roles
- **[Cache Management](cache-management.md)** - Cache operations and monitoring

## Admin Responsibilities

### User Management
- Create and manage user accounts
- Assign roles (Admin, Curator, Public)
- Monitor user activity
- Handle access requests

### Data Curation
- Review gene staging candidates
- Trigger pipeline updates
- Monitor data quality
- Manage data sources

### System Maintenance
- Cache management and optimization
- Log monitoring and analysis
- Performance monitoring
- System health checks

## Quick Access

- **Admin Panel**: `http://localhost:5173/admin` (development)
- **API Docs**: `http://localhost:8000/docs` (development)
- **Logs**: Via admin panel or system_logs table
